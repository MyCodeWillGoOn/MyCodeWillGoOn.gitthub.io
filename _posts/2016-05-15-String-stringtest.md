#### 字符串全排（无重复）：

Permutation("123".toCharArray(),3,1);
N个数全排 N! = N*(N-1)*….
则，第一个位置上n个数有 n种。第二个位置上；剩下的n-1个数，有n-1中


```java

public static void Permutation(char a[],int size,int n){
		if(n==size-1){
			print(a);			
			return ;
		}
		for(int i=n;i<size;i++){
			swap(a,i,n);
			Permutation(a,size,n+1);
			swap(a,i,n);
		}
}

```

#### 字符串全排（有重复）：


```java
public static void Permutation(char a[],int size,int n){
		if(n==size-1){
			print(a);			
			return ;
		}
		for(int i=n;i<size;i++){
			//将a[i]与 [n,i)之间的数比较
			if(isDuplicate(a,n,i)){ 
				continue;
			}				
			swap(a,i,n);
			Permutation(a,size,n+1);
			swap(a,i,n);
		}
}

```
